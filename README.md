## Dockerize the API
Build the Docker Image
```bash

```
Run the Container
```bash
docker run -p 8000:8000 diabetes-prediction-model
```
Deploy to Kubernetes
```bash
kubectl apply -f diabetes-prediction-model-deployment.yaml
```
